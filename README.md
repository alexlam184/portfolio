- About
- Blog

## Installation

Install portfolio with `yarn`

```bash
  yarn install
  cd portfolio
```

### Linting

ESLint

```bash
  yarn lint
```

### Formatting

```bash
  yarn format
```

## Running in development mode

Start the server

```bash
  yarn dev
```

Open http://localhost:3000 with your browser to see the result.
